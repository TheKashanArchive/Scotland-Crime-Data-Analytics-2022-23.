# Scotland Crime Data Analytics (2022–2023)

A Power BI-based analytical project exploring crime and offense patterns across Scotland for the years 2022–2023.

---

## Overview

This project analyzes crime and offense datasets across Scotland using Power BI to identify regional patterns, trends, and categorical distributions.

The goal is to simulate real-world public safety analytics using interactive dashboards and data visualization techniques.

---

## Key Objectives

- Analyze crime distribution across Scottish regions  
- Identify trends in violent, property, and regulatory offenses  
- Build an interactive multi-page Power BI dashboard  
- Enable geographic-level exploration of crime data  
- Present insights through structured visual storytelling  

---

## Dataset

- Source: Kaggle  
- Link: https://www.kaggle.com/datasets/keezywallace/scottish-crimeoffences-in-2223  
- Type: Structured crime & offense dataset (2022–2023)  
- Tools used: Power BI, Microsoft Excel (preprocessing)

## Dashboard Breakdown

### Page 1 – Overview

- Interactive map of Scotland with regional data points  
- High-level summaries of crime and offense categories  
- Quick comparison across different offense types  

---

### Page 2 – Crime Data Exploration

- Breakdown of violent, sexual, and property-related crimes  
- Visuals include bar charts, line graphs, and pie charts  
- Designed for deep analytical exploration of crime patterns  

---

### Page 3 – Offense Data Exploration

- Focus on public order, regulatory, and societal offenses  
- Detailed categorical analysis using bar and pie charts  
- Highlights frequency trends and behavioral patterns  

---

## Data Filtering & Interactivity

- Custom slicer functioning as a search bar (with magnifying glass UI)  
- Region-based filtering across all dashboard pages  
- Fully synchronized cross-page filtering system  
- Interactive map enabling real-time geographic drilldowns  
- Dynamic visuals update based on user selection
